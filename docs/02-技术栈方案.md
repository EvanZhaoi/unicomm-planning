# UniComm 技术栈方案

> **重要：UniComm 是企业级桌面协作应用，不是普通 Web 管理后台。**

| 层级 | 技术 | 版本 | 说明 |
|------|------|------|------|
| **桌面容器** | Tauri | 2.x (最新稳定版) | 桌面应用框架 |
| **前端框架** | React | 最新稳定版 | UI 层 |
| **构建工具** | Vite | 最新稳定版 | 快速构建 |
| **语言** | TypeScript | 最新稳定版 | 类型安全 |
| **状态管理** | Zustand | 最新稳定版 | 轻量级状态管理 |
| **数据获取** | TanStack Query | 最新稳定版 | 服务端状态 |
| **样式** | TailwindCSS | 最新稳定版 | 原子化 CSS |
| **Markdown** | React Markdown | 最新稳定版 | 渲染 Markdown |
| **后端框架** | Spring Boot | 最新稳定版 | 支持 Java 21 |
| **语言** | Java | 21 (LTS) | 最新 LTS 版本 |
| **ORM** | MyBatis Plus | 最新稳定版 | 简化 CRUD |

## 二、桌面端技术详解

### 2.1 Tauri 2

**选型理由：**

- Rust 后端，性能优异
- WebView2 渲染，无 Chromium 包袱
- 体积小（< 10MB 安装包）
- 支持系统托盘、多窗口
- 内置自动更新
- 本地文件系统和剪贴板 API
- 支持 WebSocket

**核心能力：**

```typescript
// 窗口管理
import { getCurrentWindow } from '@tauri-apps/api/window';
const win = getCurrentWindow();
await win.setAlwaysOnTop(true);
await win.minimize();

// 系统托盘
import { TrayIcon } from '@tauri-apps/api/tray';
// 通知
import { sendNotification } from '@tauri-apps/plugin-notification';
// 文件系统
import { readTextFile, writeTextFile } from '@tauri-apps/plugin-fs';
```

### 2.2 React + TypeScript

**选型理由：**

- 成熟的 UI 生态
- Hooks API 简化状态管理
- TypeScript 保证类型安全
- 组件化开发模式
- 与 Tauri 集成良好

**编码规范：**

```typescript
// 正确：使用函数组件 + Hooks
const MemoList: React.FC<Props> = ({ memos, onSelect }) => {
  const [selectedId, setSelectedId] = useState<string | null>(null);
  // ...
};

// 禁止：class component
class MemoList extends React.Component { ... }
```

### 2.3 Zustand（状态管理）

**选型理由：**

- 轻量级（比 Redux 简单）
- 足够满足状态管理需求
- TypeScript 支持好
- 支持持久化

**按领域拆分 Store：**

```typescript
// stores/memo.store.ts
interface MemoStore {
  memos: Memo[];
  currentMemo: Memo | null;
  setCurrentMemo: (memo: Memo | null) => void;
}

// stores/socket.store.ts
interface SocketStore {
  connected: boolean;
  reconnecting: boolean;
  connect: () => void;
  disconnect: () => void;
}

// stores/session.store.ts
interface SessionStore {
  user: User | null;
  token: string | null;
  login: (user: User, token: string) => void;
  logout: () => void;
}

// stores/settings.store.ts
interface SettingsStore {
  theme: 'light' | 'dark' | 'system';
  notifications: boolean;
  setTheme: (theme: 'light' | 'dark' | 'system') => void;
}
```

### 2.4 TanStack Query

**选型理由：**

- 服务端状态管理（列表、详情）
- 自动缓存 + 后台刷新
- 离线支持
- 加载状态、错误处理

**使用示例：**

```typescript
// 正确
const { data, isLoading, error } = useQuery({
  queryKey: ['memos', { groupId, keyword }],
  queryFn: () => memoApi.list({ groupId, keyword }),
  staleTime: 1000 * 60 * 5,
});

// 与 WebSocket 配合：数据更新后主动 invalidate
const queryClient = useQueryClient();
socket.on('memo:updated', () => {
  queryClient.invalidateQueries({ queryKey: ['memos'] });
});
```

### 2.5 TailwindCSS

**选型理由：**

- 原子化 CSS，快速开发
- 减少 CSS 文件体积
- 一致性高
- 支持深色模式

**配置：**

```js
// tailwind.config.js
module.exports = {
  content: ['./index.html', './src/**/*.{js,ts,jsx,tsx}'],
  darkMode: 'class',
  theme: {
    extend: {
      colors: {
        primary: '#2563EB',
      }
    }
  }
}
```

## 三、后端技术详解

### 3.1 Spring Boot 3

**选型理由：**

- Java 21 支持（虚拟线程、Pattern Matching）
- 简化配置
- 内嵌 Tomcat/Jetty
- 生态成熟

### 3.2 WebSocket

**用途：**

- 实时消息推送
- 在线状态同步
- 通知推送

**实现方式：**

```java
@ServerEndpoint("/ws")
public class WebSocketServer {
    @OnMessage
    public void onMessage(String message, Session session) {
        // 处理消息
    }
}
```

### 3.3 Sa-Token 认证

**选型理由：**

- 轻量级
- 支持 Token 认证
- 支持分布式 Session
- 权限鉴定灵活

## 五、技术风险与备选方案

| 风险 | 应对方案 |
